# Organizador de Arquivos em Python

Projeto desenvolvido em Python para organizar arquivos automaticamente de acordo com suas extensões.

## Funcionalidades

- Criação automática de pastas
- Organização de documentos (.docx)
- Organização de PDFs (.pdf)
- Organização de imagens (.jpg)
- Organização de vídeos (.mp4)
- Organização de arquivos de texto (.txt)

## Tecnologias utilizadas

- Python
- os
- shutil
